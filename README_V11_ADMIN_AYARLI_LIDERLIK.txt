REAL MEDYA AGENCY V11 - ADMIN AYARLI LİDERLİK

Bu sürümde canlı/admin paneline Ajans Tablosu Ayarları eklendi.

Admin panelinden ayarlanabilir:
- Ana sayfa liderlik etiketi
- Ana sayfa liderlik başlığı
- Liderlik açıklaması
- Liderlik sayfası başlığı/açıklaması
- Sıralama modu: yayın saatine göre otomatik veya admin sıra numarasına göre
- Tablo tema: Neon Premium / Gümüş Premium / Sade Koyu
- TikTok kullanıcı adını göster/gizle
- Sıra rozeti göster/gizle
- Şampiyon/elit etiketi göster/gizle
- 1. sıra neon çerçevesi aç/kapat
- Yayıncı sıra numarası
- Takma ad, TikTok adı, Instagram adı, profil görseli
- Yayın saatini başlat/durdur ve +1s / -1s düzenleme

Başlatma:
START_SITE.bat dosyasına çift tıkla.

Site:
http://localhost:3000

Admin/yayıncı yönetimi:
http://localhost:3000/canli-admin.html
