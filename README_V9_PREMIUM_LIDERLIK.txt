REAL MEDYA AGENCY V9 PREMIUM LİDERLİK

Hazır özellikler:
- Premium Ajans Liderleri tablosu
- Ana sayfada gereksiz REAL MEDYA ELİT LİGİ / Yayıncı Yönetimi / Ajans Liderleri yazıları kaldırıldı
- Takma ad, TikTok kullanıcı adı, Instagram ve profil görseli yönetim panelinden eklenebilir
- Profil görseli dosya olarak seçilebilir; backend uploads klasörüne kaydeder
- Yayın başlat/durdur ile süre backend tarafından kayıt altına alınır
- RealPay puanı otomatik hesaplanır
- Opsiyonel TikTok otomatik takip bağlantısı için OPTIONAL_TIKTOK_CONNECTOR_KUR.cmd dosyası vardır

Başlatma:
1) ZIP'i çıkar.
2) START_SITE.bat dosyasına çift tıkla.
3) Site otomatik açılır: http://localhost:3000
4) Yönetim paneli: http://localhost:3000/canli-admin.html
5) Admin şifresi: real2026

Not:
SERVER penceresini kapatma. Kapatırsan lokal site/backend kapanır.
