REAL MEDYA AGENCY V8 - KOKTEN BASLATMA COZUMU

Bu surumde start sorunu kokten cozuldu:
- npm install gerekmez.
- express/cors/dotenv kurulumu gerekmez.
- Sadece Node.js yeterlidir.
- START_SITE.bat tek tikla server penceresini acar ve siteyi otomatik tarayicida acar.
- Server penceresi acik kalmalidir.

KULLANIM:
1) ZIP'e sag tikla > Tumunu ayikla.
2) Klasorun icine gir.
3) START_SITE.bat dosyasina cift tikla.
4) Site otomatik acilir.

Adresler:
Ana site: http://localhost:3000
Yayinci Yonetimi: http://localhost:3000/canli-admin.html
Admin sifresi: real2026

Hata olursa:
- SERVER yazan siyah pencerenin fotografini at.
- logs/start-log.txt dosyasinin fotografini at.
- logs/server-output.txt dosyasinin fotografini at.

Otomatik TikTok LIVE connector opsiyoneldir:
- Site ve liderlik paneli npm olmadan acilir.
- TikTok otomatik baglanti denemesi icin OPTIONAL_TIKTOK_CONNECTOR_KUR.cmd calistirilabilir.
- TikTok bu baglantilari zaman zaman engelleyebilir/degistirebilir; bu yuzden panelde manuel baslat/durdur da vardir.
